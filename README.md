# Keyboard-Tester

a simple Keyboard testing tool written in Python

## Install the Keyboard Library

```
pip install keyboard
```

## How to use

* **Clone the repository** 

```
git clone https://github.com/adithyasandew/Keyboard-Tester.git
```
* **Change Directory**

```
cd Keyboard-Tester
```
* **Run Python Script**

```
python keytest.py
```
